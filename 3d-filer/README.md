# 3D filer
Filer i 3D format (stl, blend...) skal plasseres her.

## Organisering
Siden dette prosjektet er basert på flere bevegende deler, burde vi organisere disse inn i egne subfoldere.
For eksempel burde rotorfilene bli plassert under ```3d-filer/rotor/```.

I tillegg kan det være et poeng i å organisere nye og gamle versjoner i sine egne subfoldere.
For eksempel burde gamle rotorfiler bli plassert under ```3d-filer/rotor/old/```
Omvendt blir ```3d-filer/rotor/new/```
